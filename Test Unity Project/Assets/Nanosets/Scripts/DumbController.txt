Тупой контроллер управления персонажем
======================================

Тривиальный контроллер приемущественно для неигровых персонажей, содержит
в себе минимум логики и анимаций.

### Действия и анимации

DumbController поддерживает всего 3 анимации:

1. idle - состояние ожидания
2. walk - движение
3. attack - атака
4. fall - падение (не обязательно)

Аниматор должен поддерживать следующие параметры:
bool walk
  true - персонаж идет
  false - персонаж стоит на месте
bool falling
  true - персонаж падает
  false - персонаж стоит или идет
триггер attack для запуска анимации атаки
