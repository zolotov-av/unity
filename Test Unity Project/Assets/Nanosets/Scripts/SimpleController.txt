Простой контроллер управления персонажем
========================================

Данный контроллер содержит минимум функций, минимум анимаций и позволит
управлять почти любой моделью персонажа. Для анимации персонажа вам
потребуется всего 3-4 анимации (стоит, бежит, атака, падение)

### Действия и анимации

SimpleController поддерживает всего 3 анимации:

1. idle - состояние ожидания
2. walk - движение
3. attack - атака
4. fall - падение (не обязательно)

Аниматор должен поддерживать следующие параметры:
bool walk
  true - персонаж идет
  false - персонаж стоит на месте
bool falling
  true - персонаж падает
  false - персонаж стоит или идет
триггер attack для запуска анимации атаки
